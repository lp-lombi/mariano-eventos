# Sitio Eventos (base)

Proyecto mínimo con:

- Frontend: React + Vite + Tailwind
- Backend: Node + Express

## Ejecutar

1. Backend:
   - ir a carpeta `backend`
   - `npm run dev`

2. Frontend:
   - ir a carpeta `frontend`
   - `npm run dev`

Frontend en `http://localhost:5173` y backend en `http://localhost:3001`.
